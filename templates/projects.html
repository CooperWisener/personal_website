Attack or Contain

GitHub

Oct 2025 – Dec 2025

My submission to the NFL's Big Data Bowl. A defensive analytics project using change point detection to identify how defensive players react to pass plays in zone coverage. Built a multiclass classification model with XGBoost to determine whether a defender should attack the pass catcher or play to contain the receiver.

R XGBoost Change Point Detection NFL Analytics

Pass Catcher CPOE

GitHub

Sept 2025 – Oct 2025

Modeled expected completion percentage (xCP) for NFL pass catchers using Random Forests via R's ranger package. Applied xCP to rank receivers by Completion Percentage Over Expected (CPOE), determining which receivers outperform or underperform their expected production.

R ranger Random Forests NFL Analytics

4th Down Simulation using Markov Chains

GitHub

Oct 2025 – Nov 2025

Built a Markov Chain to model state transitions in NFL drives, then used it to simulate 4th down decision outcomes. Compared Green Bay Packers head coach Matt LaFleur's 4th down tendencies against league averages to evaluate decision-making efficiency.

R Markov Chains Simulation NFL Analytics
